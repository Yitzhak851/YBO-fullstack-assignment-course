# User Profile Web App
This is a small full-stack web application built <br>
for a Full Stack course assignment. <br>
The app displays a user profile, including: <br>
- Name
- Profile picture
- Username
- Email
- Bio
- List of posts
<br> This is the assignment from week number 1

# The instructions of this assignment:
Build a small web app that shows a user's profile.
It should display user's:
•	name
•	picture
•	email
•	bio
•	list of posts.
Use Python with Flask for the backend and HTML+CSS+JavaScript for the frontend.
You can use these free APIs for this task:<br>
- JSONPlaceholder (fake users and posts) [https://jsonplaceholder.typicode.com](https://jsonplaceholder.typicode.com)
- DiceBear (cartoon avatar illustrations) [https://www.dicebear.com](https://www.dicebear.com)

## Technologies Used
- Python
- Flask
- HTML
- CSS
- JavaScript

## APIs Used
- JSONPlaceholder - fake users and posts
- DiceBear - cartoon avatar illustrations
