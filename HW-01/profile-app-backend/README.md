# Profile App Backend
This folder contains the backend code for the profile app assignment. <br>
The backend is built using Python and Flask, and it provides an API endpoint to create a user
profile. <br>
The API endpoint accepts a POST request with a JSON body containing the name, age, and bio of the profile. <br>
The backend validates the input and returns the created profile as a JSON response. <br>
To run the backend, navigate to this folder in the terminal and run the command "python app.py". <br>
Make sure you have Flask installed in your Python environment. You can install it using pip if you haven't already: <br>
```bash
pip install Flask
```
