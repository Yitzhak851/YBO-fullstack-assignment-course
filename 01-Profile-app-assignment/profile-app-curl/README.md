# User Profile App

A small full-stack web app built with Flask for the backend and HTML, CSS, and JavaScript for the frontend.

## Features
- Displays user name
- Displays user picture
- Displays user email
- Displays user bio
- Displays list of posts
- Uses JSONPlaceholder for user and post data
- Uses DiceBear for avatar generation

## Run locally
```bash
pip install -r requirements.txt
python app.py
```

Then open:
```bash
http://127.0.0.1:5000
```
